# Hash module

## hash:sha256

Generates a SHA256 of a given string.

**Parameters**
- `data`: a string to hash

**Return value** `String`: hexadecimal hash

**Author**
- [@SuperFola](https://github.com/SuperFola)

## hash:md5

Generates a MD5 of a given string.

**Parameters**
- `data`: a string to hash

**Return value** `String`: hexadecimal hash

**Author**
- [@SuperFola](https://github.com/SuperFola)
